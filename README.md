# ✨ Magic To-Do Web App

## Project Overview

The **Magic To-Do Web App** is an interactive task management application built using **HTML5, CSS3, and Vanilla JavaScript**.

This application helps users organize their daily tasks by allowing them to add, complete, edit, and delete tasks. It includes a magical animated interface with glowing effects, smooth transitions, and local storage support so tasks remain available after refreshing the page.

---

## 🚀 Features

- ✅ Add new tasks instantly
- ✅ Display newly created tasks in the Pending Tasks section
- ✅ Mark tasks as completed
- ✅ Automatically move completed tasks to the Completed Tasks section
- ✅ Edit task descriptions
- ✅ Delete tasks permanently
- ✅ Pending task counter
- ✅ Completed task counter
- ✅ Task creation timestamp
- ✅ Completed timestamp tracking
- ✅ Data persistence using localStorage
- ✅ Empty state messages
- ✅ Responsive design for desktop and mobile
- ✅ Magical animations and glowing UI effects

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Browser LocalStorage API
